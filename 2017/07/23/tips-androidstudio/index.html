问题一：为什么出现Unsupported major.minor version 52.0？

在网上找了一圈，最后在stackoverflow找到了本质原因

简单来说，就是java的编译环境版本太低，java 8 class file的版本是52，Java 7虚拟机只能支持到51。所以需要升级到java 8 vm才行。
